HTML Tag Description Use
Header Is a container for metadata (data about data) and is placed between the html tag and the body. Represents a container for introductory content or a set of navigational links.
Nav Tag defines a set of navigation links. Purpose is to provide navigation links, either within the current document or to other documents.
Aside Element represents a portion of a document whose content is only indirectly related to the document's main content. Indicating content that is related to the surrounding content within an article or web page.
Article Represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable Most commonly used to contain information that may be distributed independently from the rest of the site or application.
Section Is a thematic grouping of content, typically with a heading. Defines sections in a document, such as chapters, headers, footers, or any other sections of the document.
Time Element represents either a time on a 24-hour clock or a precise date in the calendar. This element is for presenting dates and times in a machine readable format.
Footer Defines a footer for a document or a section. Contains information about the author of the section, copyright data or links to related documents.
Img Element embeds an image into the document The tag for putting pictures on your web page
